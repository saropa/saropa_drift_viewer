DriftDebugErrorLogger

DriftDebugErrorLogger class abstract final

Best-practice error and message logger for the Drift debug server.

Uses developer.log for errors (with optional stack traces) so IDEs and debugging tools can display them. The error callback is defensive (never throws) so it is safe to use from catch blocks.

Example:

DriftDebugServer.start(
  query: runQuery,
  onLog: DriftDebugErrorLogger.logCallback(prefix: 'DriftDebug'),
  onError: DriftDebugErrorLogger.errorCallback(prefix: 'DriftDebug'),
);

Kept as a class (not an extension) for stable public API. Static methods take prefix as first arg; kept as class for API stability.
