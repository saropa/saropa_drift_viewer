# Feature 31: Regression Test Generator

## What It Does

Snapshot the current database state and auto-generate Dart/Flutter test files that verify the data relationships, constraints, and expected values. Infers assertions from FK relationships, null patterns, value distributions, and row counts. Output ready-to-run `flutter test` files.

## Architecture

### Assertion Inferrer

Analyzes live data to determine which assertions are meaningful:

```typescript
type AssertionType =
  | 'rowCount'
  | 'fkIntegrity'
  | 'notNull'
  | 'unique'
  | 'valueRange'
  | 'exactSnapshot';

interface IAssertion {
  type: AssertionType;
  table: string;
  column?: string;
  sql: string;                    // The SQL that verifies this assertion
  expectation: string;            // Human-readable: "should be 0" or "should equal 1250"
  reason: string;                 // Why this assertion was inferred
  confidence: 'high' | 'medium'; // How likely this is a real invariant
}

class AssertionInferrer {
  constructor(private readonly _client: DriftApiClient) {}

  async infer(
    tables: string[],
    enabledTypes: Set<AssertionType>,
  ): Promise<IAssertion[]> {
    const assertions: IAssertion[] = [];
    const meta = await this._client.schemaMetadata();

    for (const tableName of tables) {
      const table = meta.tables.find(t => t.name === tableName);
      if (!table) continue;

      // Row count
      if (enabledTypes.has('rowCount')) {
        assertions.push({
          type: 'rowCount',
          table: tableName,
          sql: `SELECT COUNT(*) AS cnt FROM "${tableName}"`,
          expectation: `equals ${table.rowCount}`,
          reason: `Current row count is ${table.rowCount}`,
          confidence: 'high',
        });
      }

      // FK integrity
      if (enabledTypes.has('fkIntegrity')) {
        const fks = await this._client.tableFkMeta(tableName);
        for (const fk of fks) {
          assertions.push({
            type: 'fkIntegrity',
            table: tableName,
            column: fk.from,
            sql: `SELECT a."id" FROM "${tableName}" a LEFT JOIN "${fk.table}" b ON a."${fk.from}" = b."${fk.to}" WHERE b."${fk.to}" IS NULL AND a."${fk.from}" IS NOT NULL`,
            expectation: 'is empty',
            reason: `FK: ${tableName}.${fk.from} → ${fk.table}.${fk.to}`,
            confidence: 'high',
          });
        }
      }

      // Not-null inference
      if (enabledTypes.has('notNull')) {
        for (const col of table.columns) {
          if (col.pk) continue;
          const result = await this._client.sql(
            `SELECT COUNT(*) AS cnt FROM "${tableName}" WHERE "${col.name}" IS NULL`
          );
          const nullCount = (result.rows[0] as { cnt: number }).cnt;
          if (nullCount === 0 && table.rowCount > 0) {
            assertions.push({
              type: 'notNull',
              table: tableName,
              column: col.name,
              sql: `SELECT COUNT(*) AS cnt FROM "${tableName}" WHERE "${col.name}" IS NULL`,
              expectation: 'equals 0',
              reason: `Currently 0% null (${table.rowCount} rows)`,
              confidence: 'high',
            });
          }
        }
      }

      // Uniqueness inference
      if (enabledTypes.has('unique')) {
        for (const col of table.columns) {
          if (col.pk) continue;
          const result = await this._client.sql(
            `SELECT COUNT(DISTINCT "${col.name}") AS dist, COUNT("${col.name}") AS total FROM "${tableName}"`
          );
          const row = result.rows[0] as { dist: number; total: number };
          if (row.dist === row.total && row.total > 1) {
            assertions.push({
              type: 'unique',
              table: tableName,
              column: col.name,
              sql: `SELECT "${col.name}", COUNT(*) AS cnt FROM "${tableName}" GROUP BY "${col.name}" HAVING cnt > 1`,
              expectation: 'is empty',
              reason: `All ${row.total} values are unique`,
              confidence: row.total > 10 ? 'high' : 'medium',
            });
          }
        }
      }

      // Value range
      if (enabledTypes.has('valueRange')) {
        for (const col of table.columns) {
          if (!/INT|REAL|FLOAT|DOUBLE/i.test(col.type)) continue;
          const result = await this._client.sql(
            `SELECT MIN("${col.name}") AS mn, MAX("${col.name}") AS mx FROM "${tableName}" WHERE "${col.name}" IS NOT NULL`
          );
          const row = result.rows[0] as { mn: number; mx: number };
          if (row.mn != null) {
            assertions.push({
              type: 'valueRange',
              table: tableName,
              column: col.name,
              sql: `SELECT * FROM "${tableName}" WHERE "${col.name}" < ${row.mn} OR "${col.name}" > ${row.mx}`,
              expectation: 'is empty',
              reason: `Current range: [${row.mn}, ${row.mx}]`,
              confidence: 'medium',
            });
          }
        }
      }
    }

    return assertions;
  }
}
```

### Dart Test Renderer

Converts assertions to Dart test file source:

```typescript
class DartTestRenderer {
  render(assertions: IAssertion[]): string {
    const lines: string[] = [
      '// Generated by Saropa Drift Advisor — verify database invariants.',
      '// Modify assertions as needed for your use case.',
      '',
      "import 'package:flutter_test/flutter_test.dart';",
      '',
      '/// Test helper: runs a raw SQL query against the test database.',
      '/// Replace this with your actual database access pattern.',
      'Future<List<Map<String, dynamic>>> query(String sql) async {',
      "  // TODO: Replace with your database query method",
      "  throw UnimplementedError('Wire up your test database');",
      '}',
      '',
      'void main() {',
    ];

    // Group assertions by type
    const groups = this._groupBy(assertions, a => a.type);

    const groupNames: Record<AssertionType, string> = {
      rowCount: 'Row counts',
      fkIntegrity: 'FK integrity',
      notNull: 'Null constraints',
      unique: 'Uniqueness',
      valueRange: 'Value ranges',
      exactSnapshot: 'Exact data',
    };

    for (const [type, groupAssertions] of groups) {
      lines.push(`  group('${groupNames[type]}', () {`);

      for (const assertion of groupAssertions) {
        const testName = this._testName(assertion);
        lines.push(`    test('${testName}', () async {`);
        lines.push(`      final result = await query(`);
        lines.push(`        ${this._dartString(assertion.sql)},`);
        lines.push(`      );`);

        if (assertion.expectation.startsWith('equals ')) {
          const val = assertion.expectation.replace('equals ', '');
          lines.push(`      expect(result.first['cnt'], ${val});`);
        } else if (assertion.expectation === 'is empty') {
          lines.push(`      expect(result, isEmpty,`);
          lines.push(`        reason: '${this._escDart(assertion.reason)}');`);
        }

        lines.push('    });');
        lines.push('');
      }

      lines.push('  });');
      lines.push('');
    }

    lines.push('}');
    return lines.join('\n');
  }

  private _testName(a: IAssertion): string {
    switch (a.type) {
      case 'rowCount': return `${a.table} has ${a.expectation.replace('equals ', '')} rows`;
      case 'fkIntegrity': return `every ${a.table} references a valid ${a.column}`;
      case 'notNull': return `${a.table}.${a.column} is never null`;
      case 'unique': return `${a.table}.${a.column} is unique`;
      case 'valueRange': return `${a.table}.${a.column} stays within range`;
      default: return `${a.table} ${a.type}`;
    }
  }
}
```

## Wiring in extension.ts

```typescript
context.subscriptions.push(
  vscode.commands.registerCommand('driftViewer.generateTests', async () => {
    const meta = await client.schemaMetadata();
    const tables = meta.tables
      .map(t => t.name)
      .filter(n => !n.startsWith('sqlite_'));

    const selected = await vscode.window.showQuickPick(
      tables.map(t => {
        const info = meta.tables.find(m => m.name === t)!;
        return { label: t, description: `${info.rowCount} rows`, picked: info.rowCount < 10000 };
      }),
      { canPickMany: true, placeHolder: 'Select tables to test' }
    );
    if (!selected || selected.length === 0) return;

    const types = await vscode.window.showQuickPick([
      { label: 'Row counts', value: 'rowCount', picked: true },
      { label: 'FK integrity', value: 'fkIntegrity', picked: true },
      { label: 'Null constraints', value: 'notNull', picked: true },
      { label: 'Uniqueness', value: 'unique', picked: true },
      { label: 'Value ranges', value: 'valueRange', picked: false },
    ], { canPickMany: true, placeHolder: 'Select assertion types' });
    if (!types || types.length === 0) return;

    await vscode.window.withProgress(
      { location: vscode.ProgressLocation.Notification, title: 'Generating tests…' },
      async () => {
        const inferrer = new AssertionInferrer(client);
        const assertions = await inferrer.infer(
          selected.map(s => s.label),
          new Set(types.map(t => t.value as AssertionType))
        );

        const renderer = new DartTestRenderer();
        const dartCode = renderer.render(assertions);

        const doc = await vscode.workspace.openTextDocument({
          content: dartCode,
          language: 'dart',
        });
        await vscode.window.showTextDocument(doc);
      }
    );
  })
);
```

## Testing

- `assertion-inferrer.test.ts`:
  - Row count assertion has correct SQL and expected value
  - FK integrity assertion generates correct LEFT JOIN query
  - Not-null: columns with 0 nulls generate assertion; columns with nulls don't
  - Uniqueness: columns where distinct = total generate assertion; others don't
  - Value range: numeric columns generate min/max bounds; text columns don't
  - Empty table → no not-null or uniqueness assertions (not enough data)
  - PK columns are skipped for not-null and uniqueness (always true)
- `dart-test-renderer.test.ts`:
  - Generated code is valid Dart syntax
  - Tests are grouped by assertion type
  - SQL strings are properly escaped in Dart
  - Empty assertions → only boilerplate, no test groups

## Known Limitations

- Generated tests use raw SQL via a `query()` helper — user must wire it up to their DB
- Row count assertions are fragile — any data change breaks them
- Inference requires data in the tables — empty tables produce no assertions
- Not-null inference doesn't know about actual schema constraints — infers from current data
- Uniqueness inference may flag columns that happen to be unique in test data but aren't constrained
- No support for Drift's type-safe API — generated tests use raw SQL only
- Value range assertions don't account for legitimate future data outside current range
- Generated file needs manual import adjustments for the user's project structure
